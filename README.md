**AVIF Thumbnail Handler** is a native C++ COM Shell Extension that adds AVIF thumbnail preview support to Windows 10 Explorer.

**AVIF 缩略图处理器** 是一个原生 C++ COM Shell 扩展，为 Windows 10 资源管理器添加 AVIF 缩略图预览功能。
